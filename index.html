    외부 클라우드 API 의존 없이 보안이 유지되는 로컬 환경(Ollama) 내에서, AI가 스스로 생각(Thought)하고 도구(Action)를 사용해 결과를 관찰(Observation)할 수 있도록 ReAct 형태의 프롬프팅 및 실행 루프 구조를 직접 설계하고 구현했습니다.

동작 흐름

사용자 요청 → ReAct 에이전트가 상황을 분석하고 도구를 선택 → 시스템 프롬프트(AGENTS.md) 규칙 검증 → 파일 수정 등 도구 실행 → 센서(Sensor)가 동작하여 문법 오류를 검사 → 피드백 루프를 거쳐 최종 응답을 도출합니다.
